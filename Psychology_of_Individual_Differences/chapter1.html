Personality research methods


성격의 정의

social skill and effectiveness / individual's dominant feature or trait

인간의 모든 정신적 조직 / 인간의 본질 / 개인의 지속적이고 독특한 감정, 사고, 행동에 기여하는 심리적 특성

Goals of personality psychology

How People [Think], [Feel], [Behave]

성격이론이 다루는것 - 모든 사람들은...
  • 모든 사람들과 같다(The Human Nature Level / Describes people in general)
  • 어떤 사람들과 같다(Individual differences / Differences between groups)
  • 모든 사람들과 다르다(People are never exactly the same / Individuals are unique)

Type of data

    make inference based on different indicators -> clues

  • SELF-PEPORT DATA (S DATA)

    • Ask the person for their own opinion(설문지, 인터뷰, 삶 이야기)
    • 장점 : 자신에 대해 잘 앎 / 많은 정보 / 어떤경우 이 방법 유일 / 단순하고 쉬움
    • 단점 : 거짓말 / 오류(ex 오기억) / 편향

  • INFORMANT DATA (I DATA)

    • Someone observes the person in the laboratory or in daily life(그 사람에 대해 잘 아는사람에게 질문, 사진이나 비디오를 부호화)
    • 장점 : 많은 정보 / 상식 / S data보다 덜 편향될 수 있음
    • 단점 : 정보가 제한됨(ex 숨겨진 정보) / 오류 / 편향

  • LIFE OUTCOME DATA (L DATA)

    • Actual life outcomes(성격의 결과로 볼 수 있음)
    • 장점 : 본질적 중요성(실제 결과를 예측) / L data의 심리적 관련성
    • 단점 : Multi-determination(여러 요인이 life outcomes들에 영향)

  • TEST OF BEHAVIORAL DATA (B DATA)

    • Come from experimental procedures or standardized measures with objective rules for scoring performance /
      Can be collected in natural or artificial setting(Field studies vs experimental studies in lab)
    • 장점 : 객관성(왜곡이나 편향 없음)
    • 딘점 : 데이터 수집이 어려움 / 어떤 행동이 성격과 연관되어 있는지 밝혀야 함 / 해석의 어려움

  Q: 가장 좋은 데이터는? A: 상황에 따라 다름, 그러나 여러가지 데이터를 수집하는게 서로간 단점 보완 및 깊은 분석 가능


Reliability

정의 : 측정의 안정성(시간, 문항, 평가자) / 높을수록 좋은 측정 / Observed score = True score + error

신뢰도 하락 <- 잘못된 측정 도구(모호, 광범위 등) / 맥락요인(참가자, 실험자, 환경 등)

신뢰도 향상시키는 법: 신중한 측정도구 선택 / 중요한 변수 측정(ex 맥락 변수 영향이 적은 것 선택) / 집계(평가를 평균화)

Validity

정의 : the extent to which a test measures what it is supposed to measure

종류 : Construct validity / Face validity / Criterion validity / Convergent validity / Discriminant validity

Generalizability

정의 : 일반화 가능성은 연구 결과나 테스트의 한계와 범위를 의미 / 표본에서 나온 연구결과를 모집단에 적용하는 것이 얼마나 가능한가의 문제

Generalizability over participants - Gender Bias / Show Versus No-Show / Ethnic and cultural Diversity

Reserch designs

  • Case method

    • In depth analysis of individual cases
    • 장점 : 그 사람에 대해 깊게 이해 / 어떻게 마음이 동작하는지에 대한 새로운 아이디어를 발생시키는데에 좋음
    • 단점 : 일반화가 안될 수 있음 / 문제의 원인을 파악하지 못할 수 있음

  • Experimental method

    • Experiment - 통제된 환경 / 무작위 표본 / 통제집단과 실험집단
    • 장점 : 환경을 통제할 수 있음 / 변인을 조작 가능 / 인과를 추정 가능
    • 단점 : 인공적인 환경 / 불가능하거나 비윤리적일 수 있음

  • Correlational method

    • A way to find and describe relationships among measured variables(ex 두 변수 go together)
    • [Correlation coefficient]로 측정됨 (심볼 'r') -1 < r < 1 / 양수-정적상관, 음수-부적상관
    • 장점 : 수 많은 변수간의 상관관계를 알 수 있다.
    • 단점 : 개인을 깊게 연구할 수 없다 / 증가된 질문지의 신뢰도, 타당도 문제 / 인과관계 측정 불가능

Inferring causation

상관관계 != 인과관계

세가지 가능성 : a -> b / b -> a / c -> a&b

ecological validity

실험실 환경 != 실제 세계
