심리학의 개관


심리학은 어떤 학문인가?

정의: 인간의 정신과정과 행동에 관한 과학적 연구


심리학의 연구방법

설문조사 연구: 설문지 개발 -> 연구변인 양적으로 수치화 -> 관련성 추정

사례연구: 개인, 소수를 관찰, 면접 -> 결론도출

실험 연구: 통제환경에서 변인 조작 -> 영향분석

심리학의 역사

심신이원론(마음 몸 별개) vs 심신일원론(마음 몸 하나의 실체)


구조주의 심리학

  • 마음을 구성요소로 분석, 구성요소간 상호작용을 연구
  • 정신을 구성하는 요소와 내용을 밝히기 위한 시도
  • 분석적 내성법: 정신을 구성하는 심상, 느낌, 감각을 분석하는 방법

기능주의 심리학

  • 구조주의에서 내성법이 불가능한 대상을 연구에서 제외함을 비판
  • 인간이 갖고 있는 의식의 내용 < 심리적 기능 :
    의식이나 정신과정은 여러 개의 요소로 나뉠 수 없는 하나의 상태. 의식의 전체적인 기능을 밝혀야 한다고 주장 의식의 기능, 의식이 존재하는 목적 이나 이유에 관심

형태주의 심리학

  • 의식의 내용을 구성 요소로 분석하면 원래의 내용은 사라지고, 이 요소를 다시 결합한다 해도 원래대로 돌릴 수 없다고 주장
  • 마음이나 의식을 잘 조직화된 하나의 전체로 이해해야 한다고 주장 (전체는 부분의 합 이상의 것)

행동주의 심리학

  • 왓슨: 관찰 가능한 사건 또는 행동만을 연구, 정신과정을 배제
  • 파블로프: 인간의 모든 행동은 조건형성의 결과, 자극과 반응의 관계로 규정

정신분석

  • 공격성과 관련된 무의식적 충동 -> 인간행동
  • 프로이트 - 무의식을 심리학에 포함
  • 아동 초기의 경험들이 성격발달에 가장 중요한 요인

현대 심리학의 관점

  • 행동주의적 관점

    : 환경자극에 의해 인간의 행동이 결정된다는 접근법

  • 정신분석적 관점

    : 인간의 행동을 본능과 무의식으로 설명

  • 생리심리학적 관점

    : 생리적 작용으로 인간의 심리적 사실을 설명

  • 인지적 관점

    : 기억, 지각, 학습, 지능, 언어, 문제해결을 집중적으로 연구

  • 인본주의적 관점

    : 인간의 자유의지와 자아실현에 대한 욕구
