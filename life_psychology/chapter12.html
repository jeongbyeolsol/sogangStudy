이상행동에 대한 이해

이상과 정상

“이상”의 판단기준

  • 통계적 일탈

    집단평균에서 어느 정도 벗어났느냐에 따라 구분

  • 사회적·문화적 규준의 위반

    개인이 문화의 행동양식이나 기준에 부합되느냐에 따라 구분

  • 주관적 고통

    개인이 주관적으로 얼마나 심리적 고통을 느끼느냐에 따라 구분

  • 위험성

    : 유기체에 대한 생물학적이거나 진화론적인 불이익을 초래하게 되면 장애로 보는 입장


이상 분류의 판단 : DSM-5

  • 분류의 장점 : 전문가 간 의사소통 촉진, 원인과 치료법에 대한 연구 가능

  • 분류의 단점 : 낙인과 편견


판단을 위한 필요 정보

  • 행동관찰        
  • 주호소/문제행동        
  • 사회적 기능수준        
  • 지지자원

이상행동의 분류

정신증(psychosis)
  • 현실검증력 없음
  • 표현하는 증상을 이해하기 어렵고 쉽게 공감할 수 없음
  • 자 아 기 능 의 퇴행이 심하게 나타남
신경증(neurosis)
  • 현실검증력 있음
  • 표현하는 증상을 이해할 만하고 공감할 수 있음
  • 주관적 고통감이 큼(자아이질적)
성격장애(personal disorder)
  • 현실검증력 있음
  • 표현하는 증상을 이해할 만하고 공감할 수 있음
  • 자아동조적

이상행동의 설명모델

정신분석적 접근

대표 학자: 프로이트 / 무의식적인 갈등과 내적 결핍의 표현 / 성격구조: 원초아, 자아, 초자아

이상행동 :
균형와해 -> 불안 유발 -> 갈등과 해결되지 못한 문제들이 무의식 속에 묻히게 됨 -> 유사한 스트레스, 갈등 시 의식에 잠재되어 있던 문제가 다양한 방식으로 반복 경험 -> 정신장애 발생


인간중심적 접근

대표학자: 로저스(Rogers)

인간은 누구나 자아실현과 성장이라는 목표를 향해 끊임없이 변화하려는 의지가 있고, 환경과의 상호작용을 통해 자기 자신을 확인

이상행동 : 주관적으로 경험하는 낯설거나 새로운 자극이 이미 조직화되어 있는 개인의 자기구조 안으로 동화되지 못하거나 불협화음을 일으킬 때, 갈등과 부적응 문제가 발생

경험 != 자기개념

행동주의적 접근

이상행동 - 학습의 결과 or 정응행동을 습득하지 못한 결과

고전적 조건화: 혐오자극과 무해한 자극 연합 -> 불안 / 조작정 조건화: 이상행동이 강화 / 관찰학습: 이상행동을 모방


인지적 접근

인지적인 정보처리과정의 오류에 의해 발생 / 인지행동적 치료법이 개발중


생물학적 접근

정신장애가 신체가 기능하는 과정상의 이상에 기인 / 광범위하며 유전적이고 생화학적인 요인에 초점


소질-스트레스 모형 접근(diathesis-stress model)

개인의 특성과 환경의 상호작용을 강조 / 개인의 소질이나 취약성

정신장애가 특정한 유전적 소인이나 뇌신경계의 이상을 지닌 개인이 살아가면서 특정한 스트레스에 노출 시 기인

이상행동의 실제

우울장애

유병률과 실태

  • 유병률은 청소년기를 거치면서 현저히 증가
  • 여성은 청소년 초기부터 시작하여 남성보다 1.5배~3배 가량 높은 비율로 경험
  • 회복이 되면 기능 수준이 병전으로 완전히 되돌아가기도 함
  • 성격장애, 불안장애, 물질사용장애를 가지고 있을 경우에는 증상이 만성화
  • 주요우울장애 -> 양극성장애로 발전하는 경우도 있음 ( 특히 사춘기에 발병하거나 정신병적 양상 이나 양극성장애 가족력이 있을 경우)
  • 자살 행동 가능성이 높은데, 가장 위험한 요인은 과거 자살 시도력

원인론 및 치료

  • 생물학적 이론

    신경전달물질의 역할 이상

    약물 치료

  • 정신역동적

    외부로 향하는 공격적 추동/분노가 자신을 향함

    심리 상담 : 자유연상 -> 해석 -> 재검토

  • 인지 이론

    자신과 세계에 대한 왜곡된 인지도식에 기인

    심리 상담 : 역기능적 사고 검토, 사고 오류 교정

  • 행동주의이론

    통제할 수 없는 사건에 대한 반응으로 무력감이 학습

    심리 상담 : 바람직한 행동 보상, 우울행동에 채계적 무시 -> 능력 향성


양극성장애

[DSM-5 진단기준] : 적어도 1회의 조증 삽화를 만족

조증삽화 : 비정상적으로 들뜨거나, 의기양양하거나,과민한 기분. 그리고 목표지향적 활동과 에너지의 증가가 적어도 일주일간 거의 매일, 하루 중 대부분 지속되는 분명한 기간이 있다

유병률 (평생 유병율: 0.4~1.6% / 평균 연령은 약 18세.)

원인과 치료

  • 생물학적 요인: 유전적 요인(가족력) / 신경전달물질

    치료 : 약물치료 / 전기충격치료

  • 심리적 요인 : 깊은 우울의 반동형성

    치료 : 심리상담


공황장애


주의력결핍 과잉행동장애(ADHD)

상담에 대한 실제적 질문

Q : 제 문제가 상담을 받을 만한 문제인지 모르겠어요

  • 진단을 위한 상담: 내 문제에 대한 진단을 받는 것만으로도 도움이 될 수 있음
  • 성장을 위한 상담: 심리상담은 개인의 심리적 성장을 위한 공통적 목표. 정신질환이 있는 사람만이 받는 서비스가 아님
