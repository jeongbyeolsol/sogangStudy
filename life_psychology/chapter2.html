발달과 변화


발달심리학이란

발달적 변화는 성숙(maturation)과 학습(learning)의 산물, 한 개인의 전 생애에 걸쳐 일어나는 신체적, 사회적 정서적, 도덕적 및 지적 발달의 과정을 연구

발달심리학자의 목적: 기술, 설명, 최적화 / 규준적 발달과 개별적 발달을 이해 -> 적정 시기의 적절한 개입

발달심리학의 주요 논점

  1. 발달의 연속성(연속성 VS 단계) : 발달이 점진적인지 혹은 급격한 변화를 동반하는 과정인지의 문제
  2. 발달의 원천(천성 VS 양육) : 발달이 유전에 의한 것인지 혹은 환경에 의한 것인지?
  3. 개인차의 문제(안정성 VS 변화) : 사람 간의 차이가 안정적인지 그리고 그 원인이 무엇인지의 문제

발달심리학의 연구방법

횡단연구

: 연령이 다른 집단들을 동시에 비교

종단연구

: 한 집단의 사람이 발달해 가는 과정을 추적하는 연구기법.

인간 발달의 영역

  • 생물학적 발달 - 신체적 변화
  • 인지적 발달 - 사고, 지능, 언어에서의 변화
  • 사회정서적 발달 - 대인관계, 정서, 성격의 변화

신체발달

  1. 유아기
  2. 취학 전 아동기
  3. 학령기
  4. 청소년기

Piaget의 인지발달 단계이론

모든 지적 활동은 사고 과정과 주변 환경 사이의 균형, 즉 인지적 평형(cognitive equilibrium)을 이루려는 목표를 가짐

도식 : 특정한 유목에 속하는 대상이나 상황에 대한 반응

평형(도식과 경험간 조화) ->동화(기존의 도식을 근거하여 해석) -> 조절(기존의 도식을 수정) -> 조직화(기존의 도식을 재정리)

인지발달의 단계

  1. 감각 운동기: 감각과 운동능력을 통해 세상을 탐구하고 정보를 습득 대상영속성 발달
  2. 전조작기: 언어와 정신적 표상, 상징을 사용하기 시작, 그러나 사고는 직관적이고 자아 중심적, 보존과제x
  3. 구체적 조작기: 구체적 사물과 사상들에 대해 조작, 보존과제o
  4. 형식적 조작기: 추상적이고 가설적인 상황에 대해서도 사고 가능
Piaget의 인지발달 단계에 근거한 청소년기 특징

추상적 개념(감각을 통해서 직접 경험될 수 없는 개념)에 대한 사고능력(ex 사회인지 : 다른 사람의 감정이나 생각, 의도 등을 이해하는 능력)

사고과정에 대한 사고능력(ex 사고과정에 대한 사고능력 / 자의식적 사고)

    자의식적 사고 -> 상상적 청중 / 개인적 우화

Freud의 심리성적 성격발달이론

  1. 구순기[0~1]
    리비도가 입에 집중 -> 구강경험 추구 / 욕구불만 -> 비관적인 성격특성 / 고착 -> 손가락을 빰
  2. 항문기[1~3]
    항문 주위에 집중 -> 긴장 생성 -> 배설 -> 완화 / 배변훈련 / 관대하고 창의적이며 생산적인 성격을 발달
    고착 -> 항문기 강박적 성격 : 강박적이고 결벽증적으로 인색하며 지나치게 인내 / 항문기 폭발적 성격 : 지저분하고 무책임한 성향
  3. 남근기[3~5]
    오이디푸스 콤플렉스 -> 동일시 -> 초자아가 발달
  4. 잠재기[5~사춘기 이전]
    리비도가 집중되는 신체의 특정 부위가 없는 비교적 평온한 시기 / 인지발달, 문화적 사회적 가치 수용
  5. 성기기[청소년기]
    되살아난 성적 충동과 현실 간의 갈등문제 / 리비도의 대상 - 이성 친구 / 욕구 억제하고 현실에 대처하는 자아 강해짐 자신의 쾌락추구보다는 타인을 배려하는 성격이 강해짐

Erikson의 심리사회적 성격발달이론

  1. 유아기 - 신뢰 대 불신 : 신뢰를 갖기 위해 일관성과 따뜻함 중요
  2. 아동기 - 자율성 대 수치심 : 엄마와 나는 분리된 존재
  3. 학령전기 - 주도성 대 죄책감 : 사회적 규범을 내재화
  4. 학령기 - 근면성 대 열등감 : 갈등이 쉬어가는 시기
  5. 청소년기 - 정체감 확립 대 역할 혼비 : 아이와 어른 사이. 심리적 갈등으로 인한 불안정한 시기
  6. 청년기 - 친밀감 대 고립감 : 사랑이란? 서로의 다름을 인정, 혼자임을 인정
  7. 장년기 - 생산성 대 침체감 : 살아온 삶에 대한 점검. 다음세대에 대한 생산성
  8. 노년기 - 자아통합 대 절망감 : 자신의 생애와 화해하는 시기
